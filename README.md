# CoffeePriest
Code from CoffeePriestBot on Telegram


This Telegram bot automates the management of coffee duty rotations within a group. It allows users to define participants, schedule who is responsible for making coffee, and set the duration of each assignment period. The bot keeps the rotation organized and ensures responsibilities are distributed fairly among a dynamic number of users.


Feel free to use and/or improve it to you needs.
